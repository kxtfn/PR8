Як у Python відкрити і прочитати файл?

Прізвище: Денисенко
Відповідь на питання першого студента:
У Python файл відкривають за допомогою функції open(ім'я, режим).
Найзручніше використовувати конструкцію with ... as, бо файл закриється автоматично.
Основні режими: 'r' – читання, 'w' – перезапис, 'a' – дописування в кінець.
Після відкриття можна використовувати write()/writelines() для запису або read()/readline() для читання.
Помилки (немає файлу, немає прав) бажано ловити через try-except.

Прізвище: Лихацький
Відповідь на питання другого студента:
Для перевірки існування файлу в Python найчастіше використовують модуль os:
import os
if os.path.exists('ім'я_файлу.txt'):
    print('Файл існує')
else:
    print('Файл не знайдено')
Щоб прочитати файл пострічково, найефективніше це зробити у циклі for:
try:
    with open('ім'я_файлу.txt', 'r', encoding='utf-8') as f:
        for line in f:
            print(line.strip()) # .strip() прибирає зайві символи переносу рядка
except FileNotFoundError:
    print('Помилка: Файл для читання не знайдено.')
