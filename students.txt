Питання для наступного студента:
Як у Python відкрити і прочитати файл?
---
Прізвище: Денисенко
Відповідь на питання першого студента:
У Python файл відкривають за допомогою функції open(ім'я, режим).
Найзручніше використовувати конструкцію with ... as, бо файл закриється автоматично.
Основні режими: 'r' – читання, 'w' – перезапис, 'a' – дописування в кінець.
Після відкриття можна використовувати write()/writelines() для запису або read()/readline() для читання.
Помилки (немає файлу, немає прав) бажано ловити через try-except.
Питання для наступного студента:
Як у Python перевірити існування файлу та прочитати його пострічково?
Відповідь:
---
Прізвище: Лихацький
Відповідь на питання другого студента:
Для перевірки існування файлу в Python найчастіше використовують модуль os:
import os
if os.path.exists('ім'я_файлу.txt'):
    print('Файл існує')
else:
    print('Файл не знайдено')
Щоб прочитати файл пострічково, найефективніше це зробити у циклі for:
try:
    with open('ім'я_файлу.txt', 'r', encoding='utf-8') as f:
        for line in f:
            print(line.strip()) # .strip() прибирає зайві символи переносу рядка
except FileNotFoundError:
    print('Помилка: Файл для читання не знайдено.')
Питання для наступного студента:
Яка різниця між методами `append()` та `extend()` при роботі зі списками в Python?
Відповідь:
---
Прізвище: Іващенко
Відповідь на питання третього студента:
Різниця між append() та extend() у Python полягає в тому, як саме вони додають елементи до списку.
append():
 1. Додає один елемент у список.
 2. Якщо передати список — він буде доданий як один елемент (вкладений список).
extend():
 1. Додає кожен елемент з переданого iterable окремо.
 2. Не створює вкладений список — "розгортає" елементи.
Питання для наступного студента:
Що означає *args, **kwargs та як вони використовуються?
Відповідь:
---
Прізвище: Гордієнко
Відповідь на питання четвертого студента:
*args, **kwargs - це спеціальні механізми в Python, які дозволяють функції приймати змінну кількість аргументів.
*args збирає всі позиційні аргументи у кортеж, тому функція може отримати будь-яку кількість значень без чіткого опису кожного параметра.
**kwargs збирає всі іменовані аргументи у словник, що дає можливість передавати пари 'ключ=значення' у довільній кількості.
Використовуються вони тоді коли потрібно зробити функцію гнучкою - наприклад, писати універсальні обробники, будувати розширювані класи або передавати у функцію невідоме наперед число параметрів.
